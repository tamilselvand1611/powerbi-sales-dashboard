# Superstore Sales Dashboard (Power BI)

## Tool
- Power BI Desktop

## Dataset
- Sample Superstore Dataset (Kaggle)

## Project Description
Created an interactive Power BI dashboard to analyze sales and profit trends across product categories, sub-categories, and regions.

## Key Insights

### Overall Performance
- Total Sales: 2.30M
- Total Profit: 286.40K
- Profit Margin: 12.5%

### Sales by Category
- Technology leads in total sales.
- Furniture and Office Supplies follow closely.

### Sales Trend
- Overall sales show a steady increase over the years.

### Sales by Sub-Category
- Phones and Copiers are the top-performing sub-categories.
- Tables show relatively lower sales.
- Some sub-categories contribute to lower or negative profits compared to others.

## Key Features
- KPI cards for Sales, Profit, and Profit Margin
- Category and sub-category level analysis
- Trend analysis over time
- Interactive slicers for better exploration
